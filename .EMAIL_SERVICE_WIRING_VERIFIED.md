## Key Features (Brevo-Only)

### Email Quota Management
- Daily limit: 300 emails/day (configurable via `BREVO_DAILY_LIMIT`)
- When quota exceeded:
  - All running campaigns auto-pause
  - Reason saved as: `pauseReason: 'quota_exceeded'`
  - UI displays message to user
  - Campaigns resume next day or when quota resets

### Campaign Status Tracking
- Emails logged with metadata: `{ provider: 'brevo', messageId: ... }`
- Events recorded in MongoDB via CampaignLog.record()
- Frontend displays send progress and email delivery stats

### Template Support
- Dynamic token replacement: `{{customer_name}}`, `{{business_name}}`, `{{review_link}}`, `{{feedback_link}}`
- HTML email generation from campaign template text
- CTA button styling (configurable via campaign settings)
